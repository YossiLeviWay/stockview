Creating StockView Project

This project is centered around creating a comprehensive and insightful financial dashboard. This dynamic dashboard serves as a centralized hub, presenting a range of key financial metrics and statements. With an emphasis on clarity and user-friendly visualization, the dashboard provides an intuitive interface for users to analyze and monitor various aspects of their financial data.


I utilized the BeautifulSoup library in Python to gather and scrape data from various financial websites. This powerful tool allowed me to automate the process, enabling me to collect massive amounts of data with minimal effort. By leveraging automation, I could efficiently retrieve and organize the data for further analysis.


Excel played a crucial role in my data workflow, as it provided a platform for data cleaning and organization. Through Excel, I was able to apply various techniques to cleanse and structure the data, ensuring its accuracy and usability. Additionally, I harnessed the power of automation using Visual Basic to save time and create patterns in the data. This allowed me to work more efficiently and consistently with the data.


To handle multiple data sources and create a comprehensive database, I turned to SSIS (SQL Server Integration Services). This tool enabled me to gather data from various sources and seamlessly integrate it into a unified database. With SSIS, I could efficiently manage and update the database as needed, ensuring its accuracy and relevance.


SSMS (SQL Server Management Studio) became an essential part of my workflow, as it facilitated the creation of jobs and facilitated data updates at specific times. By leveraging SSMS, I could automate routine tasks and schedule data updates, enhancing the overall efficiency of my data management process.


Power BI and DAX (Data Analysis Expressions) were instrumental in analyzing and presenting data in a visually appealing and insightful manner. With Power BI, I could create measures and calculations to analyze different aspects of the data. DAX allowed me to define complex formulas and expressions to derive meaningful insights from the data. Together, Power BI and DAX enabled me to design interactive dashboards that effectively conveyed the data's key information in a convenient and user-friendly way.
